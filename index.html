Theory & Mathematics

Understanding Convolution in Time and Frequency Domains

Time Domain

Sliding Integration

In the time domain, convolution represents the system overlap. The signal $h(t)$ is flipped, shifted by $t$, and multiplied by $x(\tau)$. The area under the product curve gives the output value at that instant.

y(t) = ∫ x(τ) · h(t - τ) dτ

For discrete signals, this becomes a sliding sum:

y[n] = Σ x[k] · h[n - k]
Frequency Domain

Convolution Theorem

The Convolution Theorem states that convolution in the time domain is equivalent to point-wise multiplication in the frequency domain. This is why we can compute it much faster using the FFT!

Y(f) = X(f) · H(f)
y(t) = F⁻¹ { X(f) · H(f) }

The dotted line in the Output panel shows this theoretical result computed via FFT.
